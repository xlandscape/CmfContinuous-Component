Installation guide¶

Get Python¶

  • Get the most recent version of Python: https://www.python.org/. Note that this installation comes with the Python core packages others must be added afterwards.
  • A common way to install additional packages in Python is using pip: https://pypi.org/project/pip/
  • Another way is to use a full development environment which ships with a user interface and the most used packages, e.g. WinPython: http://winpython.github.io/

Get dependencies¶

  • shapely (required by CMF)
  • scipy (optional; needed by pre-processing tools)
  • matplotlib (optional; needed by plotting tools)

Get models¶

  • CMF
  • Regulatory Catchment Model

An installation with pip is recommended:

1
2
  pip install cmf
  pip install LanscapeModel
